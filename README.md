# Homebrew Tap

My personal Homebrew Tap.

## Add the tap

```sh
brew tap yasanglass/tap
```

## Remove the tap

Remove the tap when you no longer need it:

```sh
brew untap yasanglass/tap
```

## Available casks

### [Orbit Compose Multiplatform Catalog](https://github.com/yasanglass/orbit-compose-multiplatform)

```sh
brew install --cask yasanglass/tap/orbit-compose-multiplatform-catalog
```

### [Kepko sample app](https://github.com/yasanglass/kepko)

```sh
brew install --cask yasanglass/tap/kepko-sample
```

## First launch

These apps are currently ad-hoc signed and not notarized by Apple. macOS may require approval under System Settings -> Privacy & Security before the first launch.
